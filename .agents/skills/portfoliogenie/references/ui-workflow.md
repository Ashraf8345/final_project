# UI Workflow

Before building UI

Step 1

Check if shadcn already provides the component.

If yes

Install it.

Do not recreate it.

Step 2

Compose multiple shadcn components.

Step 3

Only create custom components when necessary.

Step 4

Extract reusable components.

Step 5

Ensure accessibility.

Step 6

Ensure responsive layouts.

Step 7

Optimize for performance.

Never build everything inside page.tsx.
