# Animation Guidelines

Animations should improve UX.

Never distract users.

Prefer

Fade

Slide

Scale

Opacity

Small transforms

Avoid

Large rotations

Elastic bouncing

Long animations

Flashy effects

---

Framer Motion

Use

whileHover

whileTap

layout

AnimatePresence

Use spring transitions sparingly.

---

Timing

Fast

150ms

Normal

200ms

Slow

300ms

Rarely exceed 400ms.

---

Respect prefers-reduced-motion.

Animations should gracefully degrade.

---

Loading

Use Skeleton components.

Avoid spinners whenever skeletons provide a better experience.

---

Microinteractions

Hover

Focus

Pressed

Loading

Success

These should feel subtle and responsive.
